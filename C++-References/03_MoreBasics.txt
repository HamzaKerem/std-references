C uses only 32 reserved keywords, while C++ uses 95!
C++ standards: C++98, C++03, C++11, C++14, C++17, C++20

There are 3 ways of initializing variables.
C style:			double r = 10.4;
Constructor style:  double r (28.5);
C++11 style:		double r {28.5}; // Recommended way.
